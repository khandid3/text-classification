Enter the text in french you want to predict the difficulty.

This website allows you to make prediction on the difficulty of a french given text. You can insert several sentences. When you click on submit button, it will display you the difficulty of the text. The model to make prediction is based on classification concept from machine learning. Our different classes are {A1, A2, B1, B2, C1, C2}. The model has been trained with Google Cloud Natural Language. You can access to the source code on my github page.
